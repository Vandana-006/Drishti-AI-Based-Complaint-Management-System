# System Design

## Overview
This system uses AI-based facial recognition to identify missing persons from a centralized database.

## Components
1. Police Station
   - Upload FIR details and images

2. Central Database
   - Stores all missing person data

3. AI Engine
   - Matches uploaded images with database

4. Dashboard
   - Displays results to police

## Flow
Police Station → Central Database → AI Processing → Dashboard Output
